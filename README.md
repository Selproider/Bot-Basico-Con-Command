# Bot-Basico-Con-Command
Este bot es un ejemplo de bot con commands.Bot que incluye un comando de saludo $hello, un comando que dice cuándo se unió un miembro $joined @usuario, un comando divertido que repite "he" $heh 5 (o cualquier número), muestra información en consola al iniciar y usa intents para leer mensajes y obtener información de miembros.
